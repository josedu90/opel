All JS Activation and Custom JS

In this section I talk about javascript plugins activation and extra front-end javascript code. Which gives this template awesome flexibility. I will try to organized all javascript code. Hope you should understood.

All JS Function Name

Every section I added a comment which is allow you to find out any javascript code easily. Now take a look all javascript function name below list.

            
    1. slider_style_one (Carousel One Activation)
    2. slider_style_two (Carousel Two Activation. Ex. All vehicle or 
       site carousel Active with this)
    3. slider_style_three (Carousel Three Activation)
    4. brand-slider (Company brand carousel Activation)
    5. fun_fects (Fun facts count-to Activation)
    6. time_count (For Coming Soon Time Count Activation)
    7. accordion (FAQ Accordion Activation)
    8. gallery (Single Vehicle Gallery Activation)
    9. scroll_top (Scroll to top button)
    10. datepicker (Search form date and time picker)
    11. searchOverlay (Search Modal)
    12. mobileTab (Mobile Tab JS)
    13. mapIframe (Map disable zoom on Scroll)
    14. widget_mobile (All widget on mobile version)
